<h2>Projeto Java com API Rest</h2>

Projeto para testar, unitariamente, uma API REST para o gerenciamento de estoques de cerveja.

* API REST; 

* Frameworks para testes unitários em Java: JUnit, Mockito  Hamcrest;

* TDD.
